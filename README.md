### JSON

Json files on a web server



#### Want to Contribute

1. Fork this repo

2. Clone this repo

```bash
$ git clone https://github.com/flyingsonu122/JSON.git
$ cd JSON
```
3. open db.json

4. Add new Wish

```bash
{
    "wish": "write your wish here between double quotes"
}
```

```
* Do not change the format

* Do not Delete other wishes

```
5. Create Pull Request
