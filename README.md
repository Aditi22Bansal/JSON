# Diwali Wishes




### Want to Contribute?



1. Fork this repo



2. open db.json

3. Add new Wish at last 

```bash

{
    "wish": "write your wish here between double quotes"
}

```


* Do not change the format

* Do not Delete other wishes

* Do not repeat wish

6. Commit your changes 

7. Create Pull Request
